# Java 实训

# Day1

各种环境安装和介绍

安装 jdk

jdk 三大组成

1) JVM：java 虚拟机

2) jre：java 运行环境

3) jdk：java 开发工具包

安装软件注意事项

尽量不用中文目录

目录不用太深

java 环境变量配置

  1. JAVA_HOME: 配置 java 的安装目录， 即 bin 的上以及目录。 如： JAVA_HOME=D:\Java1.82\jdk 2. Path：

  2. 环境变量的配置，目的是在任何位置都可以找到要运行的命令。

    绝对路径： Path=D:\Java1.82\jdk\bin;D:\Java1.82\jdk\jre\bin;

    相对路径： Path=% JAVA_HOME%\bin;% JAVA_HOME%\jre\bin;

# 构建工具 Maven 安装

# 1. 概念

Maven 项目对象模型 (POM)，可以通过一小段描述信息来管理项目的构建，报告和文档的项目管理工具软件。

# 2. 什么是构建
  • 将 java 项目中的源码， 通过清理、编译、 测试、打包、安装、 部署整个过程称之为项目的构建。
  • 清理： 清理之前编译的内容， 如字节码文件。
  • 编译： 将 java 的源文件编译成字节码文件。
  • 测试： 对代码中的核心功能通过测试工具，如 JUnit 进行测试。
  • 打包： 将生成的字节吗文件打包为 jar 或者 war 包。
