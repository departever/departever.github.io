# 计算机系统基础

# 源码，反码，补码

原码：将最高位为符号位（0 表示正，1 表示负），其它数字位代表数值本身的绝对值的数字表示方式。

反码：如果是正数，则表示方法和原码一样；如果是负数，符号位不变，其余各位取反，则得到这个数字的反码表示形式。

补码：如果是正数，则表示方法和原码一样；如果是负数，则将数字的反码加上 1（即在反码最低位加 1）。

e.g. 6 源码 0000 0110 反码 0000 0110 补码 0000 0110
     -10 源码 1000 1010 反码 1111 0101 补码 1111 0110

# 大端和小端的区别

小端方式：数据最低有效字节放低地址单元 / 数据最高有效字节放高地址单元

大端方式：数据最低有效字节放高地址单元 / 数据最高有效字节放低地址单元

# 移位运算

无符号数：逻辑移动

高（低）位移出，低（高）位补 0，可能溢出！

有符号数：算数移动

左移：高位移出，低位补 0。可能溢出！

溢出判断：若移出的位不等于新的符号位，则溢出。
右移：低位移出，高位补符，可能发生有效数据丢失。
